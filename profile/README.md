# WAVE

The video layer for AI agents. Stream, produce, and deliver live video at scale.

**5 packages on npm:**

```bash
npm install @wave-av/sdk          # TypeScript SDK — 34 API modules
npm install @wave-av/adk          # Agent Developer Kit — 10 tools, 5 templates
npm install @wave-av/mcp-server   # MCP Server for Claude/Cursor
npm install @wave-av/create-app   # Project scaffolder
```
